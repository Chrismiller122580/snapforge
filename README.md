# SnapForge 🛠️

AI-powered vehicle customizer web app

## Stack
- Next.js 15 (App Router)
- Prisma + PostgreSQL (Vercel)
- Auth0
- Tailwind + shadcn/ui
- React Three Fiber for 3D previews
- Deploy: Vercel

## Quick Start
1. `git clone https://github.com/Chrismiller122580/snapforge.git`
2. `cd snapforge && npm install`
3. Copy .env.example → .env.local (fill Auth0, Prisma)
4. `npx prisma generate && npx prisma db push`
5. `npm run dev`

Live on Vercel soon!
